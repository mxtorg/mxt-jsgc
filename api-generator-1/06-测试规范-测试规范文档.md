## 3. 单元测试规范

### 3.1 测试框架
- JUnit 5
- Mockito
- AssertJ

### 3.2 测试类命名
```
${类名}Test
```
示例：`NamingUtilTest`、`DtoGeneratorTest`

### 3.3 测试方法命名
```
should${行为描述}_when${条件描述}
```
示例：
```java
@Test
void shouldConvertPathToMethodName_whenGivenPostMethodAndPath()
```

### 3.4 测试结构
使用Given-When-Then三段式：
```java
@Test
void shouldDoSomething() {
    // Given - 准备测试数据
    ...
    
    // When - 执行被测方法
    ...
    
    // Then - 验证结果
    ...
}
```

### 3.5 工具类测试示例

#### NamingUtilTest
```java
package org.demo.maven.util;

import org.junit.jupiter.api.Test;
import static org.assertj.core.api.Assertions.assertThat;

class NamingUtilTest {

    @Test
    void shouldConvertPathToMethodName_whenGivenPostMethodAndPath() {
        // Given
        String httpMethod = "post";
        String path = "/users/instance";
        
        // When
        String methodName = NamingUtil.pathToMethodName(httpMethod, path);
        
        // Then
        assertThat(methodName).isEqualTo("postUserInstance");
    }

    @Test
    void shouldConvertRefToClassName_whenGivenValidRef() {
        // Given
        String ref = "#/components/schemas/DemoEndpoint";
        
        // When
        String className = NamingUtil.refToClassName(ref);
        
        // Then
        assertThat(className).isEqualTo("DemoEndpoint");
    }

    @Test
    void shouldConvertUnderscoreToCamelCase_whenGivenUnderscoreString() {
        // Given
        String str = "user_name";
        
        // When
        String camelCase = NamingUtil.underscoreToCamelCase(str);
        
        // Then
        assertThat(camelCase).isEqualTo("userName");
    }
}
```

```java
package org.demo.maven.generator;

import org.junit.jupiter.api.Test;
import org.junit.jupiter.api.io.TempDir;
import java.io.File;
import static org.assertj.core.api.Assertions.assertThat;

class DtoGeneratorTest {

    @TempDir
    File tempDir;

    @Test
    void shouldGenerateDtoClass_whenGivenValidSchema() throws Exception {
        // Given

        DtoGenerator generator = new DtoGenerator();

        generator.generate(config, tempDir);
        
        // Then
```

```java
package org.demo.maven;

import org.apache.maven.it.Verifier;
import org.apache.maven.it.util.ResourceExtractor;
import org.junit.jupiter.api.Test;
import java.io.File;
import static org.assertj.core.api.Assertions.assertThat;

class ApiGeneratorMojoIT {

    @Test
    void shouldGenerateAllCode_whenExecuteMojo() throws Exception {
        // Given
        File testDir = ResourceExtractor.simpleExtractResources(
            getClass(), "/it/simple-case");
        Verifier verifier = new Verifier(testDir.getAbsolutePath());
        
        // When
        verifier.executeGoal("org.demo:api-generator-maven-plugin:1.0.0:generate-api-code");
        
        // Then
        verifier.verifyErrorFreeLog();
        File controllerFile = new File(testDir, "src/main/java/org/demo/cloud/controller/DemoEndpoint.java");
        assertThat(controllerFile).exists();
    }
}
```

### 5.3 E2E测试示例
```java
package org.demo.maven.e2e;

import org.junit.jupiter.api.Test;
import org.junit.jupiter.api.io.TempDir;
import java.io.File;
import java.nio.file.Files;
import static org.assertj.core.api.Assertions.assertThat;

class FullFlowE2ETest {

    @TempDir
    File tempDir;

    @Test
    void shouldExecuteFullFlow_whenGivenValidConfig() throws Exception {
        // 1. 准备测试环境
        File specFile = new File("src/test/resources/api-spec.json");
        File targetDir = new File(tempDir, "output");
        
        // 2. 执行插件
        ApiGeneratorMojo mojo = new ApiGeneratorMojo();
        mojo.setSpecDir(specFile.getParentFile());
        mojo.setOutputJavaDir(targetDir);
        mojo.execute();
        
        // 3. 验证生成的文件
        assertThat(new File(targetDir, "org/demo/cloud/dto/RegisterRequest.java")).exists();
        assertThat(new File(targetDir, "org/demo/cloud/controller/DemoEndpoint.java")).exists();
        assertThat(new File(targetDir, "org/demo/cloud/feign/DemoClient.java")).exists();
        
        // 4. 验证代码可编译
        CompilationResult result = JavaCompiler.compile(targetDir);
        assertThat(result.isSuccess()).isTrue();
    }
}
```

## 8. 性能测试规范

### 8.1 性能指标
- 单个配置文件生成时间 < 5秒
- 10个配置文件并发生成 < 30秒
- 内存占用 < 512MB

### 8.2 性能测试示例
```java
@Test
void shouldGenerateCodeWithin5Seconds_whenSingleFile() {
    // Given
    ApiConfig config = TestData.createSampleConfig();
    DtoGenerator generator = new DtoGenerator();
    File outputDir = tempDir;
    
    // When
    long startTime = System.currentTimeMillis();
    generator.generate(config, outputDir);
    long endTime = System.currentTimeMillis();
    
    // Then
    assertThat(endTime - startTime).isLessThan(5000);
}
```
